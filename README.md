# AI Tools Directory

Full-stack Next.js 14 directory for AI tools organized by use case.

## Stack
- Next.js 14 (App Router)
- TypeScript
- PostgreSQL (via Neon or local)
- Prisma ORM
- Tailwind CSS + shadcn/ui
- NextAuth.js (auth)
- Vercel (deployment)

## Verticals
1. E-commerce
2. Marketing/Content
3. Legal

## Structure
```
├── app/                    # Next.js app router
│   ├── (marketing)/        # Landing pages
│   ├── directory/          # Tool listings
│   ├── api/                # API routes
│   └── admin/              # Admin dashboard
├── components/             # React components
├── lib/                    # Utilities, db client
├── prisma/                 # Database schema
└── types/                  # TypeScript types
```
